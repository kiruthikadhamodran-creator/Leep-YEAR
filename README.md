Algorithm: Leap Year Checker
Step-by-step Algorithm

Step 1: Start the program
Step 2: Read the input year from the user
Step 3: Check whether the year is divisible by 400
  → If yes, print True (Leap Year)
Step 4: Otherwise check whether the year is divisible by 100
  → If yes, print False (Not a Leap Year)
Step 5: Otherwise check whether the year is divisible by 4
  → If yes, print True (Leap Year)
Step 6: Otherwise print False (Not a Leap Year)
Step 7: Stop the program

Flow Logic (Decision Order Explanation)

Check in this order:

Is year divisible by 400?
        YES → Leap Year
        NO ↓
Is year divisible by 100?
        YES → Not Leap Year
        NO ↓
Is year divisible by 4?
        YES → Leap Year
        NO → Not Leap Year
